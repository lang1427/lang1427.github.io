Vue移动端封装组件库

  • GridView(栅格布局)
  • swiper(轮播图)

    • 依赖于better-scroll插件所封装的组件

  • scroll(封装better-scroll)
  • scrollNavbar(导航横向滚动)

参数 说明 类型 可选值 默认值

Props

参数 说明 类型 可选值 默认值

                                                      </ul>

                                              name: "test",
                                              components: {
                                                Scroll
                                              },
                                              methods: {
                                                scroll(position) {
                                                  console.log(position);
                                                }
                                              }
                                            };

示例效果：

Props

参数 说明 类型 可选值 默认值
probeType 监听滚动的位置 Number 0 :不监听滚动的位置 ，1：超过一段时间监听 ， 2：滚动时监听 ， 3：实时监听（滚动结束后的惯性滚动依然监听） 0
direction 滚动方向 String vertical：纵向 | horizontal：横向 vertical
freeScroll 是否同时支持X轴和Y轴的滚动 Boolean true | false flase
pullUpLoad 是否开启上拉加载更多功能 Boolean true | false false
