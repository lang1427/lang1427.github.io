示例代码:
                                        
    <template>
        <div class="test">
            <scroll class="scroll-wrapper" :probeType="3" @scroll="scroll">
                <div>
                    <ul>
                        <li v-for="item of 100" :key="item">{{ item }}</li>
                    </ul>
                </div>
            </scroll>
        </div>
    </template>          
                                        
                                    
                                        
    import Scroll from "@/components/scroll/scroll.vue";
    export default {
        name: "test",
        components: {
            Scroll
        },
        methods: {
            scroll(position) {
                console.log(position);
            }
        }
    };
                                        
                                
                                    
    .scroll-wrapper {
        width: 100vw;
        height: 300px;
        overflow: hidden;
        background-color: orange;
    }
示例效果：

Props

参数 说明 类型 可选值 默认值
probeType 监听滚动的位置 Number 0 :不监听滚动的位置 ，1：超过一段时间监听 ， 2：滚动时监听 ， 3：实时监听（滚动结束后的惯性滚动依然监听） 0
direction 滚动方向 String vertical：纵向 | horizontal：横向 vertical
freeScroll 是否同时支持X轴和Y轴的滚动 Boolean true | false flase
pullUpLoad 是否开启上拉加载更多功能 Boolean true | false false

事件

事件名 说明 参数
pullingUp 监听是否滚动到底部（用于实现上拉加载更多） 需对pullUpLoad设置为true才起作用 -
scroll 监听滚动的位置 需要设置probeType position：滚动的x和y轴的值

实例方法

方法名 说明 参数
scrollTo 滚动到指定的位置 {Number} x 横轴坐标（单位 px） {Number} y 纵轴坐标（单位 px） {Number} time 滚动动画执行的时长（单位 ms）
refresh 重新计算 better-scroll，当 DOM 结构发生变化的时候务必要调用确保滚动的效果正常 -
finishPullUp 当上拉加载数据加载完毕后，需要调用此方法告诉 better-scroll 数据已加载 -
enable 启用better-scroll -
disable 禁用better-scroll -
更多better-scroll方法请参考 better-scroll文档
